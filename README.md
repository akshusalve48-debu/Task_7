# Task 7 - Basic Sales Summary using SQLite and Python

## Objective
To extract sales summary data using SQL inside Python and visualize revenue by product.

## Tools Used
- Python
- SQLite
- Pandas
- Matplotlib

## Steps Performed
1. Created SQLite database and sales table.
2. Inserted sample sales data.
3. Ran SQL query using GROUP BY to calculate total quantity and revenue.
4. Loaded results into pandas.
5. Visualized revenue using a bar chart.

## Output
- Printed sales summary
- Revenue bar chart saved as sales_chart.png
